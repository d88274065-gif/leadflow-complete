# LeadFlow AI CRM

Ek hi repo mein frontend aur backend dono!

## Structure
```
leadflow-ai-crm/
├── frontend/   → Vercel pe deploy hoga
└── backend/    → Railway pe deploy hoga
```

## Frontend Deploy (Vercel)
1. Vercel → New Project → is repo select karo
2. **Root Directory: `frontend`** set karo
3. Environment variables add karo

## Backend Deploy (Railway)
1. Railway → New Project → is repo select karo
2. **Root Directory: `backend`** set karo
3. Environment variables add karo (README.md dekho)
